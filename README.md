# Docker-Nginx-PHP5.6-MySQL5.6
Build image
-----------

```bash

sudo apt-get install make
```

Run container
-------------
```bash
sudo make run
```

Check Version PHP
-------------
```bash
sudo make test
```

Run bash on container (for debug)
-------------
```bash
sudo make bash
```

## Test container
http://localhost:8080/

## Remove container
sudo make clear
